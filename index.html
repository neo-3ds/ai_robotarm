USB 연결 버튼을 누르세요

📱 연결 방법 가이드

  1. 1. 모바일 기기는 반드시 인터넷에 연결 되어 있어야 합니다.
  2. 2. Chrome 브라우저에서만 동작합니다. Chrome 브라우저가 아닌 경우 화면 가이드 팝업창을 닫고 화면에 나오는 'Chrome으로 열기' 버튼을 눌러 주세요.
  3. 3. 로봇의 전원을 끈 후 데이터 케이블로 기기와 로봇을 연결 합니다.(C type 젠더 사용)
  4. 4. QR 코드를 스캔하기 전에 이미 로봇을 기기와 연결해 놓은 경우, 로봇의 전원을 끄고, USB 케이블을 분리한 후 다시 연결해주세요.
  5. 5. 기기에서 USB 포트를 인식하면 로봇이 활성화 됩니다. 이제 로봇의 전원을 켜주세요.
  6. 6. 'USB연결' 버튼을 누른 후 'USB연결' 화면의 아래쪽 카메라 영상이 나오는 곳을 클릭하면 'FT232R USB UART' 목록이 나타납니다.
  7. 7. 목록을 선택하면 로봇이 연결됩니다.
  8. 8. '동작인식' 모드와 '화면터치' 모드를 선택하여 동작해 보세요.
  9. 9. '화면터치' 모드에서 화면 오른쪽 끝 부분을 위로 드래그 하면 집게가 열리고, 아래로 드래그 하면 집게가 닫힙니다.
